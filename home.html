Some of my projects

One of my greatest qualities as an employee is the ability to pick something up and learn it quickly. I have had to learn on the fly many times throughout my career as chef and it has not only made me a more confident performer but a great teacher when given the opportunity. Recently I built four full-stack web apps in one week bursts. I am proud of my clone of instagram called WhatsForLunch, which includes things like AWS for quick uploads, infinite scroll via React modules and custom search functions for users to enjoy a smooth consistent experience.

I also have strong skills in backend server and database technologies such as Flask and Postgres which can be seen in project Jackpot Degen. It is a web app that allows poker players throughout the country to get real time user generated jackpot updates, information and poker news as well as poker room reviews using extensive CRUD actions and RSS feeds. Although I am familiar with CSS libraries such as Material UI and Bulma, all of my projects were styled and animated with vanilla CSS.

What I enjoy most about coding is the fact that if you can think it, you can build it. I made a completely unique web app called WhatsInTheSauce that includes features such as the ability to create multiple shopping lists and then track everything inside of your fridge including expiration dates and dollar values while received updated recipe recommendations based on what is inside your fridge at any given time, all on a single, interactive page. I achieved this using multiple nested React components and api calls to a custom Flask server that retrieves data from my Postgres database and also to TheMealDB API. All of the animations, transitions and UI are created with vanilla CSS which I found to be surprisingly powerful with a little finesse and creativity.
